# Valeur Consultoria - Landing Page de Diagnóstico

Landing Page de alta conversão para Valeur Consultoria, focada em Diagnóstico Comercial. Desenvolvida em React, Tailwind e Framer Motion para uma experiência premium e profissional.

## 📁 Estrutura de Imagens (IMPORTANTE)

Para que o site exiba todas as imagens corretamente, você deve subir os arquivos na pasta `public/assets/` com os seguintes nomes precisos:

- `logo.png` - Logotipo oficial da Valeur (exibido no topo e rodapé).
- `playbook.png` - Imagem de destaque do Playbook Valeur na seção Hero.
- `felipe.png` - Foto do sócio Felipe Assis.
- `luan.png` - Foto do sócio Luan Taube.

## 🚀 Tecnologias Utilizadas

- **React + Vite** - Performance e rapidez no carregamento.
- **Tailwind CSS** - Estilização moderna e responsiva.
- **Framer Motion** - Animações e micro-interações fluidas.
- **Lucide React** - Set de ícones profissionais.

## 🛠️ Como rodar o projeto localmente

1. Clone o repositório.
2. Instale as dependências: `npm install`
3. Rode o servidor de desenvolvimento: `npm run dev`
